# TailwindCSS

TailwindCSS adalah sebuah Framework CSS yang berbasis pada utility. 
TailwindCSS memiliki berbagai kumpulan fitur dan fungsi untuk melakukan customisasi css tanpa harus membuat dari awal.

Pada TailwindCSS tidak seperti Framework CSS umum lainnya, TailwindCSS tidak memiliki komponen atau style yang telah di tentukan sebelumnya.
Namun, TailwindCSS hanya menyediakan kumpulan utilitas/alat yang dapat digabungkan untuk membuat desain seperti apapun yang dibayangkan.

## Kelebihan TailwindCSS
- Fleksibilitas: Tailwind CSS memberi Anda kendali penuh atas tampilan pada halaman website.
- Kustomisabilitas: Anda dapat menyesuaikan Tailwind CSS agar sesuai dengan sistem desain yang ditentukan.
- Reusable: Class utilitas pada Tailwind CSS dapat digunakan kembali di project lain.
- Responsivitas: Tailwind CSS menyertakan varian responsif dari kelas utilitasnya, sehingga memudahkan untuk membuat halaman web responsif.
- Performa: Tailwind CSS dikompilasi menjadi file CSS yang sangat kecil, yang dapat meningkatkan kinerja halaman web Anda.

# Pengantar HTML

## Pengertian & Gambaran Dasar HTML

HyperText Markup Language atau yang disingkat menjadi HTML adalah sebuah bahasa markup yang digunakan untuk mengembangkan website.
File HTML ditandai dengan ekstensi `.html` pada akhir nama file. Contoh nama file HTML adalah `index.html`. Untuk Menjalankan file HTML dibutuhkan sebuah `Browser` atau `Peramban`. Browser akan menampilkan isi dari file HTML yang sudah dibuat atau dijalankan.

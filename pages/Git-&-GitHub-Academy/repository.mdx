#  Repository

Repository merupakan tempat penyimpanan data yang berisi file-file yang dibutuhkan untuk menjalankan aplikasi. Repository ini dapat berupa file-file yang berada di dalam komputer, atau file-file yang berada di dalam server. 

## Cara membuat repository

### 1. Membuat repository lokal

Repository lokal merupakan repository yang berada di dalam komputer. Repository lokal ini dapat dibuat dengan cara sebagai berikut:

  1. Buka aplikasi Git Bash / terminal.
  2. Pada Git Bash / terminal, buat folder untuk menyimpan repository lokal.
  3. Setelah itu, masuk ke dalam folder.
  4. Kemudian, buat repository lokal dengan perintah git init.
  5. Repository lokal berhasil dibuat.

### 2. Membuat repository di GitHub

Repository di GitHub merupakan repository yang berada di dalam server. Repository ini dapat dibuat dengan cara sebagai berikut:

  1. Buka akun GitHub.
  2. Pada bagian atas, klik tombol + dan pilih New repository.
  3. Isi nama repository, lalu klik tombol Create repository.
  4. Repository berhasil dibuat.

### 3. Menjadikan repository lokal kedalam repository di GitHub

  1. Buatlah sebuah repository di GitHub.
  <Callout type="info">
    Untuk menghindari terjadinya error ketika menambahkan remote repository, buatlah repository tanpa berisikan _README_, license ataupun file **gitignore**.
  </Callout>
  2. Copy url  repository yang tadi sudah dibuat.
  3. Pada repository local tambahkan remote dengan perintah:

      ```bash
      git remote add origin REMOTE_URL # REMOTE_URL adalah url dari repo di buat pada github
      ```
      ```bash
      git remote -v # Untuk melihat semua remote yang terdaftar
      ```
  4. Setelah remote ditambahkan lakukan push kedalam repository.
  5. Repository local sudah berhasil dimasukkan kedalam repository di Github

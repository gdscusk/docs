# Git Instalation & Configuration

## Windows

Download versi terbaru dari [git-scm.com](https://git-scm.com/) dan install sesuai petunjuknya.

## Linux

Ikuti petunjuk di [git-scm.com](https://git-scm.com/download/linux)

## Mac

Ikuti petunjuk di [git-scm.com](https://git-scm.com/download/mac)

## Konfigurasi Git

Setelah instalasi selesai, buka terminal dan ketikkan perintah berikut:

```bash
git config --global user.name "Your Name"
```

```bash
git config --global user.email "Your@email.com"
```

Ganti Your Name dengan nama anda dan email anda.

Optional: Untuk mengubah branch default menjadi **main**, ketikkan perintah berikut:

```bash
git config --global init.defaultBranch main
```
